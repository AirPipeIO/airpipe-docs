# 00 - Intro

A powerful capability of the `airpipe` binary and platform is being able to turn any of your commands and scripts into complete APIs.

We will use basic everyday commands to demonstrate this as a way for you to imagine all the more you can do.

This tutorial will cover this on a Linux / Mac machine, this can be done on Windows or anywhere the binary can run.

### Requirements

- Linux / Mac machine
- Self-hosted local `airpipe` [installation](/docs/install)
