# Authorization for your APIs

Air Pipe supports many ways for you to protect your endpoints.

We will cover a few common options for you to assess and understand.
These can be further modified, extended or even combined depending on your requirements:

- [Token / JWTs](/docs/tutorial/authorization/jwt) (JSON Web Tokens) **recommended**
- [Basic Auth](/docs/tutorial/authorization/basic-auth)
- API Keys - see the JWT examples on ways you can consider handling this
