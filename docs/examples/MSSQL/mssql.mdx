# MSSQL / SQL Server

Our current [quickstart tutorial](/docs/tutorial/database-to-api) covers other patterns to use with your database.

## Defining the database

1. Define a `globals` section
2. Add the `databases` field
3. Add a database identifier
4. Set the driver to `mssql`
5. Add the connection string

```yml reference title="MSSQL Basic"

```
