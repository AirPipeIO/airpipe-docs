# Windows

A native Windows build is on the roadmap. Today, the recommended ways to run
Air Pipe on Windows are:

- **WSL2 (Ubuntu)** — install inside a WSL2 Linux distribution and follow the
  [Linux instructions](./linux.mdx). This is the closest experience to a
  native install.
- **Container** — run the official image with Docker Desktop. See
  [Container](./container.mdx).

Once you have a running instance, continue with the Getting Started tutorial.
