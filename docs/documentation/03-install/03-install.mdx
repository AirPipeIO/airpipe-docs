# Install

  - Register an account at https://app.airpipe.io/register.
  - Check your email to verify your account and log in.
  - You may use the hosted mode in the website interface itself or setup one of the self hosted modes as below.

## Hosted (Managed)
No further setup required once you have registered and verified your account. Easy!

## Self Hosted (Unmanaged)

- Generate an API Key at https://app.airpipe.io/apikeys
