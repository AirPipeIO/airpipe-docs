# Limits

At Air Pipe we believe in providing you as much flexibility to grow your project or business.

Our transparent usage model and effecient technology stack enables us to provide you the below benefits.

Unlimited / No Limit:
- Unlimited configuration creation
- Unlimited interfaces, actions, workflow rules
- No bandwidth/egress limits
    - Fair usage applies for hosted/managed mode

Free Usage Limits:
- 2 API Keys
- 2 Roles
- Limited revision history
