# Tests

Are an array of functions which are used to validate your data. 
To use tests functions you must declare a `tests` section under your `assert`

```yml
interfaces:
  user/register:
    output: http
    method: POST
    show_error_detail: true
    actions:
      - name: Input
        input: a|body
        hide_data_on_empty: true
        run_when_succeeded:
          - UserShouldNotExist
        assert:
          error_message: "Password not strong enough"
          tests:
            - value: first
              is_not_null: true
              is_greater_than: 2
              is_less_than: 20
              regex: \S+
            - value: last
              is_not_null: true
              is_greater_than: 3
              is_less_than: 20
              regex: \S+
            - value: email
              is_not_null: true
              is_greater_than: 3
              is_less_than: 100
              regex: \S+
            - value: pass
              is_not_null: true
              is_greater_than: 9
              is_less_than: 30
            - value: pass
              regex: "[A-Z]"
              error_message: "Password should contain one uppercase character"
            - value: pass
              regex: "[a-z]"
              error_message: "Password should contain one lowercase character"
            - value: pass
              regex: "[0-9!@#]"
              error_message: "Password should contain at least one number (0-9) or symbol (!,@,#)"
```
