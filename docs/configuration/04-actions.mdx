# Actions

Actions are a set of instructions an interface will run when called. <br/>

## Options

| Name                                                                             | Type              | Description                                                                                                                                                                                           |
| -------------------------------------------------------------------------------- | ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`                                                                           | string (required) | The name of the action                                                                                                                                                                                |
| [`input`](/docs/configuration/interfaces/actions/inputs)                         | string (optional) | The data available when an interface is called <ul><li>`a\|body`</li><li>`a\|headers`</li><li>`a\|params`</li><li>`a\|<action-name>`</li></ul> <br /> Note inputs **do not** have trailing `\|` pipes |
| [`run_condition`](/docs/configuration/interfaces/actions/run-conditions)          | object (optional) | Control whether or not the action should run |
| `hide_data_on_success`                                                           | boolean(optional) | Hide data from the response payload on success.                                                                                                                                                       |
| `hide_data_on_error`                                                             | boolean(optional) | Hide data from the response payload on error.                                                                                                                                                         |
| [`database`](/docs/configuration/interfaces/actions/inputs/databases)            | string (optional) | A database which your action will interact with.                                                                                                                                                      |
| `query`                                                                          | string (optional) | The query to be run on the specified database.                                                                                                                                                        |
| `params`                                                                         | array[string] (optional)  | Array of parameters available for the actions to use                                                                                                                                                  |
| [`assert`](/docs/configuration/interfaces/actions/asserts)                       |                   | an **array of test cases** used to validate your data and inputs. See [here](/docs/configuration/interfaces/actions/asserts) for more details.                                                        |
| [`post_transforms`](/docs/configuration/interfaces/actions/transforms/functions) |                   | an **array of functions** which are used to transform your data after it is received. See [here](/docs/configuration/interfaces/actions/transforms/functions) for more details.                       |
| [`email`](/docs/configuration/interfaces/actions/inputs/email)                   |                   | A set of options to configure email responses.                                                                                                                                                        |
| `run_when_succeeded`                                                             | array[string] (optional) | Run this action if the specified action has succeeded.                                                                                                                                                |
| [`http`](/docs/configuration/interfaces/actions/inputs/http)                     |                   | A set of options for an action to make http requests to do some sort of activity                                                                                                                      |
| [`command`](/docs/configuration/interfaces/actions/inputs/command)               |                   | Run terminal commands: <ul><li>execute cli commands</li><li>cat files</li><li>install/update packages</li></ul>                                                                                       |
| `retry`                                                                          | object (optional)  | Options: <ul><li>`attempts`</li><li>`delay`</li><li>`exponential_backoff`: `true/false`</li></ul>                                                                                                      |
| `pre_log`                                                                        | array (optional)  | Options: <ul><li>`msg`</li><li>`level`</li></ul>                                                                                                                                                      |
| `post_log`                                                                       | array (optional)  | Options: <ul><li>`msg`</li><li>`level`</li></ul>                                                                                                                                                      |
| `timeout`                                                                        | string (required) | milliseconds                                                                                                                                                                                          |
