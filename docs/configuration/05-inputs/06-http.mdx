# HTTP

:::tip Full field reference
Every HTTP option (auth, TLS, pagination, multipart, proxy) with types and
examples is in the **[HTTP reference](/docs/reference/inputs-http)**, generated
from the engine schema.
:::

Supply a HTTP url for an action to use as an input.

```yml
name: http-tests
metrics_enabled: true
docs: true

interfaces:
  http/basic:
    output: http
    method: GET

    actions:
      - name: BasicHTTP
        http:
          url: https://jsonplaceholder.typicode.com/todos/
```

## Common options

`url` (required), `method`, `headers`, `body`, `bearer_auth`, `user`/`pass`
(basic auth), `digest_auth_user`/`digest_auth_pass`, `proxy`, `timeout`,
`connect_timeout`, TLS (`ca`, `root_certificates`, `accept_invalid_certs`),
`multipart`, `pagination`, and reverse-proxy passthrough (`proxy_body`,
`proxy_headers`, …). The full list with types is in the
[HTTP reference](/docs/reference/inputs-http).
