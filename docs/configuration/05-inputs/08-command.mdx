# Command

:::tip Full field reference
Every command and output-parsing field is in the
**[Command reference](/docs/reference/inputs-command)**, generated from the engine schema.
:::

The `command` input enables you to execute terminal commands.

- curl
- execute cli tools
  - terraform apply
  - package updates
- execute scripts
- linux commands
  - df
  - ps

## Options

Run a command with `run` (and optional `shell`, `timeout`, `name`), then shape
stdout/stderr with `parse` / `parse_stderr` — `data_type` (json/csv/xml/text),
`delimiter`, `split_*`, `line_start`/`line_end`, `keys`, and regex `matches`.
The full field list is in the [Command reference](/docs/reference/inputs-command).
