## Rename Attributes


Rename attributes allows you to rename attributes in an object, or multiple attributes in an array of objects.

This leverages JSONPath for selection and intends to adhered to the RFC 9535 standard.

For examples see:
- https://support.smartbear.com/alertsite/docs/monitors/api/endpoint/jsonpath.html
- https://docs.rs/serde_json_path/latest/serde_json_path/


<h3> Usage </h3>


```yaml
# rename firstname attribute in a returned object
post_transforms:
  - rename_attributes:
      $.firstname: first
```

```yaml
# wildcard match to rename username attribute in an array of objects
post_transforms:
  - rename_attributes:
      $[*].username: USER
```

<summary>Single Object Example</summary>

### Single object example
Input
```json
  {
    "id": 1,
    "name": "Leanne Graham",
    "username": "Bret",
    "email": "Sincere@april.biz"
  }
```

```yml reference title="Rename Attributes Object"
https://github.com/airpipeio/airpipe-docs/blob/main/docs/examples/_ex_ymls/_transforms_/rename_attributes-object.yml
```

Result
```json
  {
    "id": 1,
    "name": "Leanne Graham",
    "USER": "Bret",
    "email": "Sincere@april.biz"
  }
```
