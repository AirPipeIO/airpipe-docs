# Configuration Structure

Air Pipe configuration files are built with yml and follow a logical flow.

<br />

The start of a configuration file contains:

- `name`: the configuration name i.e. login-api, register-api
- `metrics-enabled:`

```yml
name: login-api
metrics_enabled: true
docs: true
```
